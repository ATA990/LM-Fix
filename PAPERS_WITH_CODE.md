# How to link your arXiv paper to official code on Papers with Code

1. Submit/claim your paper on arXiv.
2. In your arXiv author dashboard, click the **Link to code** (Papers With Code) icon.
3. Follow the flow to add this repository as **Official Code**.
4. Your arXiv page will then show an **Official Code** badge and Papers with Code will list your repo.

See: https://medium.com/paperswithcode/papers-with-code-partners-with-arxiv-ecc362883167
